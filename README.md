# english_to_tumbuka_json_datasets
This dataset is mainly designed to be used for fine-tuning Small Language Models, as it grows I believe it can be used to train LLMs as well in the future.

# Note:
If you are a Tumbuka native speaker and would like to contribute to this dataset, kindly make sure you translate the English techninal terms to a "Short Discriptive Sentence", that matches the essence of the terminology or scientific words as best as possible. I believe this will help produce better Tumbuka writing and speaking SLMs and LLMs.

# For Those that are not native speakers, but are interrested in contributing
I would suggest that you prepare a .csv file with the two columns, and upload the data here with proper naming so that people can work on it and translate it to Tumbuka.

# The Goal of this dataset:
The main goal for this dataset, is to setup a foundation, for easy translation of English into other Bantu languages and Dialects...I believe that if we can have one complete foundational dataset that is open/public in any bantu language we can easily build and compile datasets for all Bantu languages.
